# Trader-Performance-vs-Market-Sentiment
Exploratory analysis of trader performance and behavior across Fear and Greed market conditions using sentiment and trade-level data.

## Objective
The objective of this project is to analyze how market sentiment (Fear / Greed)
impacts trader behavior and performance on the Hyperliquid platform.
The goal is to uncover behavioral patterns and derive actionable trading strategies
that can inform smarter trading decisions.

---

## Datasets Used
1. **Bitcoin Market Sentiment (Fear & Greed Index)**
   - Daily sentiment data classified as Fear, Extreme Fear, Neutral, Greed, and Extreme Greed.
   - Columns include date, sentiment classification, and sentiment score.

2. **Historical Trader Data (Hyperliquid)**
   - Trade-level data containing trader account, trade size, direction (BUY/SELL),
     timestamps, and profit/loss (PnL).
